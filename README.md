# My Awesome Project 🚀

פרויקט של אפליקציות ווב אינטראקטיביות בעברית

## 📁 קבצי הפרויקט

### 🏠 [index.html](index.html)
דף הבית הראשי של הפרויקט עם ניווט לפרויקטים השונים.

### 📈 [israeli_stocks.html](israeli_stocks.html)
מעקב מניות ישראליות - ויזואליזציה אינטראקטיבית של מניות עם גרפים דינמיים.

### 🧬 [protein_folding.html](protein_folding.html)
ויזואליזציה אינטראקטיבית של קיפול חלבונים - סימולציה תלת-ממדית של תהליך הקיפול.

## 🚀 שימוש

פשוט פתחו את אחד מהקבצים HTML בדפדפן:

```bash
# פתיחת דף הבית
start index.html

# או פתיחה ישירה של אחד מהפרויקטים
start israeli_stocks.html
start protein_folding.html
```

## 🛠️ טכנולוגיות

- HTML5
- CSS3 (עם אנימציות ו-gradients)
- JavaScript (Vanilla)
- Chart.js (לויזואליזציה של גרפים)
- WebGL/Canvas (לויזואליזציה תלת-ממדית)

## 📝 הערות

כל הקבצים כתובים בעברית עם תמיכה מלאה ב-RTL (ימין לשמאל).
